# Platform-Specific Templates and Constraints

## Natural Writing Guidelines for Each Platform

### Key Principle
Write like you're actually typing the tweet/post yourself, not generating it. Use natural speech patterns, personal observations, and specific details from experience.

**Natural openers:**
- "Just figured out why..."
- "Okay so here's what happened:"
- "Been wrestling with this for weeks"
- "Funny thing about [topic]:"

**Avoid these AI openers:**
- "In today's fast-paced world..."
- "It's important to note that..."
- "Additionally, it's worth mentioning..."

Tweet 5 (Results/Impact):

[Customer quote if available]

Tweet 6 (Differentiation):

Tweet 7 (CTA):

### Single Tweet Templates

**Template 1: Problem/Solution**
```

[Solution in one line].

[Impressive metric or result].

[Link]
```

**Template 2: Comparison**
```
Before [Product]: [tedious process]

After [Product]: [simple process]

[Specific improvement metric]

Try it → [link]
```

**Template 3: Question Hook**
```
[Thought-provoking question]?

[One impressive proof point]

```

### Character Limits
- Maximum: 280 characters per tweet
- Optimal: 200-250 for readability
- Links count as 23 characters
- Images don't count against limit
